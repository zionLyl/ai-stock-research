# 财报分析

## 适用场景
- "分析茅台最新财报"
- "XX 业绩预告怎么样"
- 持仓股发布季报/年报后评估

## 工作流

### Step 1: 确认标的和报告期
- 股票代码 + 哪个季度/年度
- 是正式财报还是业绩预告？

### Step 2: 数据采集
1. **行情数据**: `python3 scripts/cn_data.py {CODE} --json`
2. **财报原文**: MCP 搜索 "{公司名} {年份} {季度}报 业绩"
3. **分析师观点**: MCP 搜索 "{公司名} 业绩 分析 研报"

### Step 3: 分析框架
| 维度 | 关注点 |
|------|--------|
| 营收 | 同比增速、环比变化、是否超预期 |
| 利润 | 净利润增速、毛利率变化、非经常性损益 |
| 现金流 | 经营性现金流、收现比、资本开支 |
| 资产质量 | 应收账款变化、存货变化、商誉 |
| 指引 | 管理层对下季度/全年展望 |

### Step 4: 输出报告
- 一句话总结：超预期/符合/不及预期
- 关键数字表格
- 对投资逻辑的影响
- 评级变化建议（加仓/持有/减仓）

## 数据源
- Sina/Tencent: 最新估值数据
- MCP 搜索: 财报公告原文、分析师研报
- 巨潮资讯(cninfo.com.cn): 官方公告（通过 MCP 搜索获取链接）

## 注意
- A股财报发布有窗口期（4月30日前出年报，8月底前出半年报）
- 业绩预告和正式财报可能有差异
- 关注非经常性损益（如卖资产、政府补贴）对利润的影响
